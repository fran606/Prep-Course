* Variables
 * Strings
 * Funciones (argumentos, `return`)
 * Declaraciones `if`
 * Valores booleanos (`true`, `false`)

variables : una variabale es un lugar donde puedo almacenar un tipo o tipos de datos,en otras palabras es un contenedor que contiene 
información a la que podemos acceder.
 Podemos pensar en una variable como una caja que contiene items, una caja que puedes abrir y cerrar y a la que le puedes añadir cosas.

strings: los llamados strings son cadenas de letras o caracteres usado para representar el texto.

funciones:una funcion es un conjunto de lineas de codigo que hacen una tarea en concreto,y esta funcion puede retornar un valor
las funciones tienen la posibilidad de tener parametros o argumentos,que son variables que modifican su comportamiento.

declaraciones: un if se utiliza para evaluar una expresion condicional,si se cumple esta condicion se ejecutara un bloque de codigo
si es falsa es posible ejecutar otras sentencias.

valores booleanos: Una valor booleano representa un valor de verdad; es decir, TRUE o FALSE. 
Una expresión booleana o un predicado booleano puede generar un valor de desconocido, que se representa mediante el valor nulo.
